Config Pelican: Themes, Disqus and Pygments

themes

First of all, search 'Pelican Themes', browse the web page, if decided, open the github page then clone the theme into

{yourpath}/blog/themes/{themeName}

Then, open 'pelicanconf.py', add

THEME = 'themes/{themeName}'

make html
make serve

reload localhost:8000, you'll see your new-blog.

note: when using windows to build pelican, you need to download 'make' in your computer and add the tool into environment variables.

disqus

Regist an account, then click 'setting' button, 'add disqus to site'.

disqus will ask for a site name and an unique disqus url. Remember!The unique disqus url is your 'shortname'.

If u forgot the shortname, log in disqus, enter https://disqus.com/admin, click 'settings', search for 'Site Identity' -> Shortname, u'll see the shortname.

Open pelicanconnf.py, add:

DISQUS_SITENAME = u"shortname"

make html
make serve

It'll be done.

pygments

pygments is the code highlights component. open:

{your path}\blog\themes\{theme name}\static\css

u'll find pygments already exist in the css file. It means u can use the tool directly.

Pygments contains a bunch of styles, open it's offical site, create one or open an arbitrarily link, it shows a dropdown box on the right side, contains all the styles. Choose one, then open git bash(windows)/terminal(mac), enter the css path of your project:

cd {your path}\blog\themes\{your theame name}\static\css

pygmentize -S xcode -f html > xcode.css

in this example i download xcode.css, then enter the basic html to change the css link from pygments.css to xcode.css, your code hightlight style can be changed.
